## Serendip 反思框架

每次完成任务后，在 `memory/reflections/YYYY-MM-DD.md` 按以下格式记录：

```
## HH:MM 反思

### Mistakes
- 做了什么：[具体操作，≥20字]
  结果：failed | partial
  教训：[可复用的经验，≥20字]

### Lessons
- 做了什么：[具体操作，≥20字]
  结果：succeeded
  收获：[可复用的经验，≥20字]
```

规则：
- 具体 > 笼统（写 "用 rm -rf 删了 /var/data" 不写 "删了文件"）
- 教训要让另一个 Agent 能直接用
- 一个操作一条，不合并
- failed 就 failed，不美化
